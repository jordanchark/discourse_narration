
Dataset Descriptions:

1. Temporal Subordinate Analysis:
- temporal_subordinate_combined.csv: Combined dataset with all temporal subordinate clauses
- temporal_subordinate_narrative.csv: Narrative-only subset
- temporal_subordinate_filtered.csv: Filtered dataset (1540-1900)

2. BÚINN and PROG Analysis:
- buinn_prog_correlation.csv: Valid data for correlation analysis
- merged_buinn_prog.csv: Merged dataset with both constructions

3. -andi Participle Analysis:
- andi_all.csv: Complete dataset for -andi participles
- andi_filtered.csv: Filtered dataset excluding 'psd' genre

4. þegar vs. þá Analysis:
- thegar_tha_combined.csv: Combined dataset for both forms
- thegar_tha_predictions.csv: Model predictions

5. Subject Position Analysis:
- subject_position_filtered.csv: Post-NT filtered data
- subject_position_narrative.csv: Narrative genre data
